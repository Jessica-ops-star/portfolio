# **App Name**: Jessica's Digital Canvas

## Core Features:

- Interactive Bio: Display Jessica's bio with smooth animations and an engaging layout to highlight her skills and experiences.
- Project Showcases: Present projects as interactive cards with descriptions, tech stacks, and real-time updates.
- Achievement Badges: Visually showcase hackathon achievements with animated badge displays for increased engagement.
- Contact Form Validation: Provide real-time validation and feedback for the contact form to improve user experience and data accuracy.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to embody sophistication and innovation.
- Background color: Dark purple (#301934), desaturated, providing a sleek dark theme.
- Accent color: Lavender (#E6E6FA), analogous, bright, to highlight key interactions.
- Headline font: 'Space Grotesk' sans-serif, modern, used for titles and section headers; body text font: 'Inter' sans-serif, providing a clean and readable text.
- Use a set of line icons that are minimalist and modern. The line weight will need to stay consistent.
- Implement glassmorphism cards for project sections to provide a modern and visually engaging experience.
- Integrate subtle animations to make sections slide in/out and to bring static UI elements to life.
